| 階段     | 內容            | 目標                                   | 案例           |
| -------- | --------------- | -------------------------------------- | -------------- |
| 第一階段 | C++基礎語法入門 | 對C++有初步瞭解，能夠有基礎編程能力    | 通訊錄管理系統 |
