## 明确C++课程学习阶段以及课程内容



| 阶段     | 内容            | 目标                                   | 案例           |
| -------- | --------------- | -------------------------------------- | -------------- |
| 第一阶段 | C++基础语法入门 | 对C++有初步了解，能够有基础编程能力    | 通讯录管理系统 |
| 第二阶段 | C++核心编程     | 介绍C++面向对象编程，为大型项目做铺垫  | 职工管理系统   |
| 第三阶段 | C++提高编程     | 介绍C++泛型编程思想，以及STL的基本使用 | 演讲比赛系统   |

* 综合大案例：机房预约系统# Blacksmith-of-CPP
